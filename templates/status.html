API Documentation

GET /gen (Combined Method)

Generate test credit card numbers using a single combined BIN string (e.g., BIN|month|year|cvv). This endpoint is under development.

"request": "curl -X GET \"http://localhost:5000/gen?bin=453201xxxxxx|12|2027|123\"",
"response_placeholder": {
  "status": "under_development",
  "message": "Combined method will be available soon. Check back for updates!"
}

GET /gen (Split Method)

Generate test credit card numbers using separate parameters for BIN, month, year, CVV, and amount. This endpoint is under development.

"request": "curl -X GET \"http://localhost:5000/gen?bin=453201xxxxxx&month=12&year=2027&cvv=123&amount=2\"",
"response_placeholder": {
  "status": "under_development",
  "message": "Split method will be available soon. Check back for updates!"
}
